# retro-utils
Little tools for retrocomputer development

# bin-to-byte.sh

Convert a charset (or any binary dumped from e.g. VICE monitor) to KickAssembler
compatible `.byte` lines

# c64charset2png.py

Convert a charset (dumped from VICE, probably) to a PNG tileset image that can
be edited in modern graphics tools.

# png2c64charset.py

Convert a PNG tileset (with a few restrictions) to a C64 charset in KickAssember
compatible `.byte` lines
